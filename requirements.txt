----------grpc evan cli-------------------
# Tải bản mới nhất
EVANS_VERSION="0.10.11"
wget https://github.com/ktr0731/evans/releases/download/v${EVANS_VERSION}/evans_linux_amd64.tar.gz

# Giải nén
tar -xzf evans_linux_amd64.tar.gz

# Di chuyển vào PATH
sudo mv evans /usr/local/bin/

# Xóa file tải về
rm evans_linux_amd64.tar.gz

# Verify
evans --version

-----------tải grpc gateway-------------------
Coi hướng dẫn trên github

sau đó tạo folder google trong folder proto, rồi tạo folder api trong folder google
Sau đó copy 4 file này của googleapis vào folder api:
google/api/annotations.proto
google/api/field_behavior.proto
google/api/http.proto
google/api/httpbody.proto

sau đó sửa file service_simple_bank.proto

sửa lại Makefile
proto:
	rm -rf pb/*.proto
	protoc --proto_path=proto --go_out=pb --go_opt=paths=source_relative \
		--go-grpc_out=pb --go-grpc_opt=paths=source_relative \
		--grpc-gateway_out=pb --grpc-gateway_opt=paths=source_relative \
		proto/*.proto

sau đó gõ make proto trong terminal
